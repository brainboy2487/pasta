# Meatballs subsystem (skeleton)
This folder contains the scaffold for the Meatball runtime, agent, and backends.
- `api/` — Rust API surface for the MRA.
- `kernel/` — optional kernel images and build artifacts.
- `runtime/` — runtime hooks and scheduler stubs.
- `agent/` — agent binary that runs inside meatballs (JSON-over-stdio).
- `backends/` — backend implementations (local, pseudo-vm, vm).
- `phase0/` — Phase 0 design artifacts.
